# Garatujas

HTML para o meu eu do futuro

no html, existem varios modos para fazer uma mesma coisa, mas so uma esta certa

divs sao como caixas, podemos mudar o tamanho e formato dessas "caixas" por meio do css

todo que esta dentro de head é meta dado e nao sera visto pelo usuario, apenas pelo navegador, com excessão do title, que aparecerá encima da página

quando usar o color, esta se referindo a cor do texto, nao do background

no css, tudo é um quadrado originalmente, para arrendondar a borda, usamos a propriedade border-radius, para fazer circulos, e só usar o border-radius: 50%;
